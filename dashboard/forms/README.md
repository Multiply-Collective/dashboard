# Form Builder Module

This module adds a custom form builder to the dashboard.

Features:

- Create and edit forms from a Google Forms-style builder.
- Add fields with field type selection.
- Choose a database mapping for each field from existing MySQL columns.
- Require login and set a minimum role for entry.
- Generate a unique public form URL for every saved form.
- Preview, edit, and delete forms from the dashboard.
- Store form submissions and mapped values.
